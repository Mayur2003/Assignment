What are the new tags added in HTML5?


The commonly used new HTML5 tags are figure, article, header, main,footer, nav, and section.

article :- Used to specify a portion of a page that includes an independent piece of content in a document, section, or website.

figure :- Used to indicate content like photos, diagrams, categorizations, maps, images, code articles, etc.

header:- Used to specify a header for a section.

main :- Used to specify the main content or important section of a document.

footer :- Used to specify a footer for a section Information like copyright, and author name are mentioned here.

nav :- Used to specify a navigation link in a webpage.

section :- Used to define a specific section in a webpage.

canvas and SVG tags


Use the HTML canvas element with either the canvas scripting API or the WebGL API to draw graphics and animations.

The exact maximum size of a canvas element depends on the browser and environment. While in most cases the maximum dimensions exceed 10,000 x 10,000 pixels, notably iOS devices limit the canvas size to only 4,096 x 4,096 pixels.

It is better to specify your canvas dimensions by setting the width and height attributes directly on the canvas elements, either directly in the HTML or by using JavaScript

SVG Tag In Html5

The svg element is a container that defines a new coordinate system and viewport. It is used as the outermost element of SVG documents, but it can also be used to embed an SVG fragment inside an SVG or HTML document.
In this example, the height and width attributes on the svg element are set using the dynamic viewport value 60vmin, equivalent to 60% of the viewport's width or height, whichever is smaller.
